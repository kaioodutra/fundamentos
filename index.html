Bem Vindo ao Git!!

Comandos Úteis



Inicializar um repositório git
- git init

Exibe o status da branch, seus arquivos e local (ex.: staging area)
- git status

Adicionar arquivos no staging area
- git add .
- git add .html
- git add index.html

Adicionar arquivos no repositório local com o commit
- git commit -m "Primeiro Commit"

Adicionar um repositório remoto
- git remote add origin "https://github.com/BrendoLima/rep_remote.git"

Empurrar para o repositório remoto
- git push origin master

Criar uma nova branch
- git branch develop

Lista as branch
- git branch

Mudar de branch
- git checkout develop
